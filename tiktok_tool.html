TikTok シート埋め & 分析ツール

ブラウザだけで完結 — TikTokアカウントのデータを取得・視覚化・シートにコピー

CORSプロキシ経由で取得を試みます。うまくいかない場合は「手動取得」をお使いください。

TikTokのプロフィールページからデータを直接コピーする方法です。100%確実に動きます。

1 TikTokで対象のプロフィールページを開く（例: @misakism13）
2 そのページで F12（またはCmd+Option+I）を押して開発者ツールを開く → Consoleタブを選択
3 以下のコードをConsoleにコピペして Enter を押す:

(() => { const s = document.getElementById('__UNIVERSAL_DATA_FOR_REHYDRATION__'); if (!s) { alert('データが見つかりません。TikTokのプロフィールページで実行してください。'); return; } const d = JSON.parse(s.textContent); const ud = d?.['__DEFAULT_SCOPE__']?.['webapp.user-detail']; const user = ud?.userInfo?.user || {}; const stats = ud?.userInfo?.stats || {}; let items = []; for (const k of Object.keys(d?.['__DEFAULT_SCOPE__'] || {})) { const m = d['__DEFAULT_SCOPE__'][k]; if (m?.itemList) { items = m.itemList; break; } } const videos = items.map(v => ({ id: v.id, desc: v.desc || '', views: v.stats?.playCount || 0, likes: v.stats?.diggCount || 0, comments: v.stats?.commentCount || 0, shares: v.stats?.shareCount || 0, duration: v.video?.duration || 0, createTime: v.createTime || 0, cover: v.video?.cover || '' })); const result = JSON.stringify({ user: { uniqueId: user.uniqueId, nickname: user.nickname, secUid: user.secUid, followers: stats.followerCount, following: stats.followingCount, hearts: stats.heartCount, videoCount: stats.videoCount }, videos }); copy(result); alert('コピー完了！(' + videos.length + '本) ツールに戻って貼り付けてください。'); })()

4 自動的にクリップボードにコピーされるので、下のテキストエリアに貼り付けてください:
